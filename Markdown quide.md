# Руководство по Markdown

## Выделение текста

Чтобы выделить текст курсивом, необходимо обрамить его звездочкой (*). Нпример, *вот так*.

Чтобы выделить текст полужирным, необходимо обрамить его двойными звездочками (**). Например, **вот так**.
